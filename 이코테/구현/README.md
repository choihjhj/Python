# [알고리즘] 구현 알고리즘

 *<h3>풀이를 떠올리는 것은 쉽지만 소스코드로 옮기기 어려운 문제*</h3>   

- 실수연산, 특정 소수점 자리까지 출력해야하는 문제
- 문자열을 특정한 기준에 따라서 끊어 처리해야 하는 문제
- 적절한 라이브러리를 찾아서 사용해야 하는 문제

보통 문자열, 완전탐색, 브루트포스, 순열조합, 스택큐 등 기본 기념문제가 구현문제로 나옴   

**[방향 벡터] 자주 활용됨** 

''' 동, 북, 서 남 으로이동 '''   
dx=[0, -1, 0, 1]    
dy=[1, 0, -1, 0]       
''' 현재위치 '''
x,y=2,2   
for i in range(4):   
    '''다음위치'''   
    nx = x + dx[i]   
    ny = y + dy[i]   
    print(nx,ny)
